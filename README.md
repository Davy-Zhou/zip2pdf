# zip2pdg-pdg2pdf
基于Python自动化解压压缩包成PDG，PDG合成PDF

仅支持zip压缩包，加密压缩包有密码本，运气好可解压

对于压缩包
用法：python zip2pdf.py D:\Test\Linux内核源代码情景分析·第1卷_10924124.zip

对于已解压目录

或者：python zip2pdf.py  D:\Test\Linux内核源代码情景分析·第1卷_10924124
